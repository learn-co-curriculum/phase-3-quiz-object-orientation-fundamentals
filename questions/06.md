# OO Syntax

Consider the following code:

```source
class Pet
  def initialize(name)
    @name = name
  end

  def name=(name)
    @name = name
  end
  
  def name
    @name
  end
end
```

```source
rabbit = Pet.new("Jabby")
rabbit.name = "Chokola"
```

After running the code in the example above, what
will&nbsp;`rabbit.name`&nbsp;return?

## Correct

Chokola

> Yes! The rabbit's name was reassigned to "Chokola".
