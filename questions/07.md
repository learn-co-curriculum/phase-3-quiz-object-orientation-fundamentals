# OO Syntax

Consider the following code:

```ruby
class Pet:

  def __init__ (self, name):
    self.name = name

buddy = Pet("Buddy")
```

What will `buddy.name` in this example output?

## Correct

```ruby
"Buddy"
```

> Correct! The value of `name` was set when the object was initialized.

## Incorrect

```ruby
TypeError
```
